구현해야할 것

1. 로그인
2. 학교 카테고리별 인증 학교 조회 (메인)
    1. 자신이 속한 학교 정보 조회 api (인증이 되었을 경우)
        - name(학교이름), imageUrl(학교이미지), address(주소)
        - 학교이미지가 없으면 기본 이미지로 (찾아서)
        - ‘상세보기’ 페이지로 학교 상세 페이지 연결처리
        - 학교 리스트 컴포넌트에 학교이름, 학교주소 포함하기
        - 동창생 리스트 보임
        - 디자인 넣기
    2. 자신이 속한 학교 정보 조회 api (인증이 안 된 경우)
        - + 학교 인증하기 (부연설명) → 내 프로필로 이동시키기
    3. 학교 이름 검색 리스트 조회 api
        - 검색창에 타이핑할 때마다 리스트가 바뀌어 표시됨. 엔터 필요없음.

1. 학교 인증하기
    1. 소재지, 학교명, 졸업년도, 졸업증명서를 입력하고 저장버튼 클릭
    2. 학교 리스트 조회 api
        1. 입력한 학교가 리스트에 있을 경우
            1. 학교 정보 연결 api
                1. 학교 유형(초중고), 학교아이디, 졸업년도까지해서 POST
                2. 알아서 이 학교에 매핑시킴.
        2. 입력한 학교가 등록되지 않았을 경우
            1. 학교 정보 생성 api
                1. 학교이름, 학교주소, 졸업년도 POST
                2. 생성하고 알아서 이 학교에 학생정보 매핑시킴.
2. 프로필 수정하기
    1. 회원정보조회 api
        1. 회원정보를 내 프로필에 디폴트로 박아둠
    2. 회원정보수정 api
        1. 유저이름, 전화번호, 주소지까지해서 PUT
